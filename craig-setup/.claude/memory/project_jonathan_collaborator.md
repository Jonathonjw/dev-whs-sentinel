Jonathan Ward (jward@epsillonmedia.com) is the founder and technical lead on Sentinal WHS. He built the majority of the codebase and manages the Railway deployment.

**Collaboration model:**
- Jonathan works directly on `master`
- Craig uses feature branches (`feature/craig-[description]`) and opens PRs for Jonathan to review and merge
- Railway auto-deploys from master — only Jonathan's merges go live

**How to apply:**
- Never push Craig's work directly to master — always use a feature branch and open a PR
- When unsure about an architectural decision, flag it in the PR for Jonathan to weigh in
- Jonathan's Claude Code workspace is the source of truth for memory and project state
